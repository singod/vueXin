# Grid 栅格

# 基础用法

栅格系统必须要row和col结合使用，row组件里面只能有col组件。
通过设置 span 可以控制每个col的长度 
<x-row >
 <x-col span="12"></x-col>
 <x-col span="12"></x-col>
</x-row>
<div style="margin-top: 15px">
<x-row>
 <x-col span="6"></x-col>
 <x-col span="6"></x-col>
 <x-col span="6"></x-col>
 <x-col span="6"></x-col>
</x-row>
</div>
<div style="margin-top: 15px">
<x-row >
  <x-col span="4"></x-col>
  <x-col span="15"></x-col>
  <x-col span="5"></x-col>
</x-row>
</div>



# Row组件的伸缩

row组件可以通过设置 flex控制整体的缩进和伸长 
<x-row flex="-50">
 <x-col span="12"></x-col>
 <x-col span="12"></x-col>
</x-row>

<x-row>
  <x-col span="12"></x-col>
  <x-col span="12"></x-col>
</x-row>
<x-row flex="150">
 <x-col span="12"></x-col>
 <x-col span="12"></x-col>
</x-row>



# Col的间隔

row组件可以通过设置 gutter控制每个col之间的距离 
<x-row gutter="10">
  <x-col span="12"></x-col>
  <x-col span="12"></x-col>
</x-row>

 <x-row gutter="20">
 <x-col span="6"></x-col>
 <x-col span="6"></x-col>
 <x-col span="6"></x-col>
 <x-col span="6"></x-col>
</x-row>



# Col的位置

row组件可以通过设置 arrange 在row组件未被撑满的情况下控制col的位置。 
 <x-row arrange="left">
   <x-col span="12">                      
   </x-col>
</x-row>

 <x-row arrange="center">
   <x-col span="12"></x-col>
</x-row>

<x-row arrange="right">
   <x-col span="12"></x-col>
</x-row>



# Col的偏移

col组件可以通过设置 offset使col向左发生偏移，如果超出24格，会整体缩进。 
<x-row arrange="left">
 <x-col  span="3" offset="3" ></x-col>
 <x-col span="3" offset="3"></x-col>
 <x-col span="3"  offset="6"></x-col>
</x-row>



# 响应式布局

col组件通过设置
pad:577px-768px np:769px-992px pc:993px-1200px wp:>1201px
可以实现响应式布局，577px以下也就是默认的移动端样式。 

<x-row  gutter="20">
<x-col  span="12"
      :pad="{span:18}"
      :pc="{span:8}"
      :wp="{span:4}">      
</x-col>
<x-col  span="12"
      :pad="{span:6}"
      :pc="{span:16}"
      :wp="{span:20}">      
</x-col>
</x-row>


 <x-row gutter="20">
 <x-col span="8"
     :pad="{span:12}"
     :pc="{span:4}"
     :wp="{span:10}">

</x-col>

     :wp="{span:6}"
