# 给图标设置loading状态

等我做了svg动画，第二种登录的动态loading效果就会更加多样好看了。当然现在是很丑的 

<x-button icon="loading" :loading="loading" @click="toggle">
加载按钮
</x-button>

export  default {
    data(){
        return{
            loading:false
        }
    },
    methods:{
              toggle(){
                  this.loading = !this.loading
              }
            },
}




# 更换图标的颜色

通过color可以更换图标颜色

<x-button icon="loading" loading color="red">红色</x-button>
<x-button icon="set" loading color="green">力量</x-button>
<x-button icon="loading" loading color="blue">蓝色</x-button>
