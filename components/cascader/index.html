# 动态数据 级联选择

# 以省市区选择为例子

使用 dynamic变为动态选择栏。
数据格式如代码所示。 还须传递一个函数:loadData="example"。 如代码所示，函数第二个参数是一个回调函数，获取的数据作为参数传给这个回调。

为保证 箭头Icon和加载Icon可以正常显示，以及选择完成后不再需要数据请求。 请为每个选项增加一个 isLeaf属性，树枝的末端设为true,如代码所示。

目前的数据只有北京>东城，西城，朝阳添加了isLeaf属性。点击其他的区则会等请求到来后再作关闭。

加载过的重复点击不会触发 loading动画 
<x-cascader :source.sync="source"
 :selected.sync="selected"
 :loadData="example"
 dynamic>
</x-cascader>


function ajax(parent_id = 0){
        return new Promise(resolve=>{
        let res = address.filter(item=>item.parent_id===parent_id)
        setTimeout(()=>{
          resolve(res)
        },1000)
        })
    }
export default {
     created(){
              ajax(0).then(res=>{
                  this.source = res
              })
            },
    data(){
        return {
            source:[]
        }     
    },
     methods:{
        example({id},updateSource){
        ajax(id).then(res=>{
        updateSource(res)
      })
      }
  },
}



# 选择即改变

使用 selectToChange，既点既显。这里发现有个问题就是再次点击选择栏，之前选的地区虽然是有选中的效果，但是并没有回到之前的高度位置，也就是要重新下拉到之前选中的地方。 后面会修复这个问题 
<x-cascader :source.sync="source"
   :selected.sync="selected"
   :loadData="example"
   dynamic
   selectToChange>
</x-cascader>



# 选择器的大小选择




使用 size控制cascader的大小，目前只有 big medium small。 默认是 medium
<x-button-group>
    <x-button @click="theSize='big'">big</x-button>
    <x-button @click="theSize='medium'">medium</x-button>
    <x-button @click="theSize='small'">small</x-button>
</x-button-group>

<x-cascader :source.sync="source"
    :selected.sync="selected"
    :loadData="example"
    dynamic
    :size="theSize">
</x-cascader>

export default {
    data(){
        return{
            theSize:'small',
        }
    }
}



# 选择器禁用选项

你只需在数据上想要禁用的地方加入 disabled:true，便可以禁用这个选项。 动态数据也是同理的。 上面这个例子 北京和 成都里面的成都小学都是不可选的 
<x-cascader :source="source" :selected.sync="selected">
</x-cascader>
